# MiniShop

Mini e-commerce full stack com React + TypeScript no front-end e Node + TypeScript no back-end.

## Estrutura

- `client`: interface React com Vite
- `server`: API Node com Express

## Scripts

- `npm run dev`: sobe front e back em paralelo
- `npm run dev:client`: sobe apenas o front-end
- `npm run dev:server`: sobe apenas o back-end
- `npm run build`: gera os builds de produção
